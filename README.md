# Caption Trans

![App Screenshot](screenshot_placeholder.png)

[中文版](README_zh.md)

A one-stop video subtitle translation tool. It extracts subtitles locally using Whisper and accurately translates them into the target language using LLM.
Currently, it supports OpenAI, Google Gemini, DeepSeek, Ollama, and other API services compatible with the OpenAI interface.
